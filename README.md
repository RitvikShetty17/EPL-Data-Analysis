
# English Premier League 2020–21 Performance Dashboard

This project presents a comprehensive interactive dashboard and data analysis of the 2020–21 English Premier League season. Built using Tableau, SQL, and Excel, the dashboard enables intuitive exploration of player and team statistics to derive performance insights for fans, analysts, and decision-makers.

## Project Overview

The EPL Performance Dashboard provides interactive visualizations that showcase:

- Top Goalscorers
- Leading Playmakers
- Team-wise Goal Contributions
- Club-wise Disciplinary Records
- Possession Analysis
- Most Impactful Players

Each component is backed by structured queries and cleaned data from the 2020–21 EPL season.

## Key Insights Generated

### 1. Top Possession-Based Teams
Analyzed passes attempted by each club to determine overall ball possession. 

### 2. Best Playmakers
Identified players with the highest average assists per match. 

### 3. Most Aggressive Teams
Clubs ranked by fouls and yellow cards committed per match. 

### 4. Top Goal Contributors
Combined goals and assists to highlight the most impactful players.

### 5. High Efficiency Players
Players with fewer than 20 matches but high G/A contributions were ranked for impact analysis


## How It Was Built

1. Data Collection & Cleaning
   Source data was cleaned using Excel for consistency, removal of nulls, and format standardization.

2. Data Analysis with SQL  
   Complex queries were written to extract insights such as:
   - Goals + assists per 90 minutes
   - Club-wise aggregate performance
   - Yellow card ratios per team
   - Average assists per match by player

3. Interactive Dashboard with Tableau  
   Cleaned data was loaded into Tableau to build dynamic dashboards allowing:
   - Filtering by club/player
   - Comparative visualizations
   - Drill-downs into performance metrics
